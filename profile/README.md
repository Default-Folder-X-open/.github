## What is this Default Folder X File Dialog Integration?
**Default Folder X** is the essential Mac utility that transforms the frustratingly limited Open and Save dialogs into powerful, customizable file navigation tools. This repository serves as a centralized resource hub for Mac users, designers, developers, writers, and anyone who saves or opens files dozens of times per day. Default Folder X adds **favorite folders**, **recent files**, **folder history**, **quick folders** (access any folder instantly), **preview**, **Spotlight search**, **rename files in save dialogs**, **folder creation**, **keyboard shortcuts**, and **app‑specific settings**. If you're tired of clicking through deeply nested folder hierarchies every time you save a file, Default Folder X is the answer. Whether you are a graphic designer saving client assets, a developer opening config files, a writer saving drafts, or anyone who files documents into organized folders, understanding the **Default Folder X file dialog pipeline** is essential for saving minutes every hour, hours every week.

## Core Features & Capabilities
- **Favorite Folders:** Add frequently used folders to a customizable menu. Access from any Open/Save dialog. Folders appear in sidebar, toolbar, or menu bar. Add with one click. Organize by project, client, or department. Works across all apps.
- **Recent Files & Folders:** Shows recently opened files and recently used folders. Quickly re‑open a file you were just working on. Jump to a folder you visited recently. Saves navigation time.
- **Folder History:** Back/forward buttons in Open/Save dialogs (like a web browser). Navigate through previously visited folders. Undo accidental navigation. No more starting from root.
- **Quick Folders:** Assign keyboard shortcuts (Cmd+1, Cmd+2, etc.) to frequently used folders. Instant navigation without mouse. Folders open with a keystroke.
- **Preview & Spotlight:** Preview files before opening (Quick Look integration). Spotlight search within Open dialogs. Find files by name, content, or metadata without leaving dialog.
- **Rename & Delete Files:** Rename files directly in Save dialog (no need to save, close, rename, re‑open). Delete files from within Open dialog. Move files to Trash.
- **Create New Folders:** Create new folders from within Save dialog. Name the folder, save file inside. No need to switch to Finder.
- **File Management:** Copy, move, duplicate files. Change file permissions. Set label colors. All from Open/Save dialogs.
- **App‑Specific Settings:** Configure Default Folder X differently per app. In Photoshop, show different favorites than in Word. In Xcode, show development folders. In Final Cut, show media folders. Settings follow the app.
- **Center & Resize Dialogs:** Open/Save dialogs appear at a default size and position (configurable). No more tiny dialogs in the corner. Dialogs remember last size and position.
- **Keyboard Navigation:** Full keyboard control. Tab through fields. Arrow keys to navigate folders. Return to select. Shortcuts for favorites.
- **Info Panel:** Shows detailed file info (size, kind, dates, permissions) before opening. Helps avoid opening wrong file.
- **Menu Bar & Finder Integration:** Optional menu bar icon (access favorites anywhere). Finder toolbar integration (open folders in Finder). Works seamlessly with macOS.

## Default Folder X Optimization Techniques
To ensure a smooth file navigation workflow, use **Favorite Folders** for your top 10‑20 directories — add them once, access them from any app. Use **Quick Folders** (Cmd+1‑9) for your most‑used folders — instant navigation. The **Default Folder X productivity pipeline** benefits from **Folder History** — use back/forward buttons to navigate like a browser. Use **App‑Specific Settings** to show different favorites in Photoshop vs Word vs Xcode. Use **Preview** before opening large files to avoid accidental launches. Use **Rename in Save Dialog** to fix typos before saving — no need to rename later in Finder.

## Supported Integrations
- **macOS:** Native integration with Open/Save dialogs, Quick Look, Spotlight, Finder, Menu Bar.
- **All Applications:** Works with every app that uses standard macOS Open/Save dialogs (including Electron apps that emulate standard dialogs). Adobe, Microsoft, Apple apps, third‑party apps.
- **File Systems:** Local drives, external drives, network drives, cloud storage (Dropbox, Google Drive, OneDrive, iCloud Drive), mounted volumes.
- **Keyboard Shortcuts:** System‑wide shortcuts (Cmd+Shift+D to show Default Folder X menu, Cmd+1‑9 for quick folders).
- **Automation:** AppleScript support (control Default Folder X from scripts). Automator actions.
